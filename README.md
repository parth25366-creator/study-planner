# Study Planner 📚

A full-stack web app for college students to track assignments, exams and deadlines.

## Tech Stack
- **Frontend:** React
- **Backend:** Node.js + Express
- **Database:** PostgreSQL
- **Auth:** JWT

## Project Structure
```
study-planner/
├── client/     → React frontend
└── server/     → Node.js backend
```

## Getting Started

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npx create-react-app .
npm start
```
